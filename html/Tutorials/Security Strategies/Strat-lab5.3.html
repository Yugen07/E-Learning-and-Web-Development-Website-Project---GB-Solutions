5.3 Use Encryption


HOW TO ENCRYPT FILES?


  • 1. In the Documentary Library, right click the Encrypted Test Folder and Select Properties from the shortcut menu.

  • 2. On the General Tab of the Properties Box, click Advanced.

  • 3. The Advanced Attributes box appears. Check Encrypt contents to secure data and click OK.

  • 4. Click OK to close the Encrypted Test Properties Box.

  • 5. A confirm attribute Change box Appears. This box indicates the attribute encrypt has been chose and asks how you want to apply this attribute. Select apply changes to this folders, subfolders and files then click OK.

HOW TO BACK TO BACK UP THE ENCRYPTION CERTIFICATE?


  • 1. Click start, eneter certmgr.msc in the Search Box and press Enter. The certmgr window opens.


  • 2. In the left pane, drill down to Personal, Certificates. In the right pane, select the certificate that shows Encrypting File System in the Intended Purposes Column. If more than one certificate is listed select them all.

  • 3. In the menu bar, click Action, All Tasks, Export. The Certificate Export Wizard appears as shown on the right side, Click Next.

  • 4. Select Yes, Export The Private key and click NEXT.

  • 5. Select Personal Information Exchange for the file format and click Next. The back up of the key will be saved.

  • 6. Create a password for your key. Click Next

  • 7. Click Browse, Navigate to where you want to save the file, name the file Encryption Key Backup and click Save.

  • 8. To complete the process, click Next, Finish and OK.

    • HOW DOES ENCYPTED FOLDER WORKS?


  • 1. In Windows Explorer, in the Documents Library what is the new color of the Encrypted Test Folder Name? What is the color of the file name in this folder?

    • NOTES: If the folder is not shown in a different color, you can fix the setting. To do so, click organize in the Window toolbar, and click Folder and Search options. In the folder options box, click the View tab, and check Show encrypted or compressed NTFS files in color.



  • 2. Verify the file in the Encrypted Test folder is encrypted.

  • 3. Copy both the Normal Test Folder and Encrypted Test Folder to the root of drive C.

  • 4. Copy the Normal Test Folder and the Encrypted Test Folder to a USB Flash drive.

  • 5. Log off, and then log on using the standard user account you created earlier in this lab.

  • 6. Open Windows Explorer and locate the Normal Test Folder in the root of Drive C. Describe what happens when you double click the file in the Normal Test Folder.

  • 7. Locate the Encrypted Test Folder in the root of drive C. Describe what happens when you double click the file in the Encrypted Test Folder.

  • 8. Log off, and then log on using your Administrator Account.

  • 9. Upon reaching the desktop, return to the encrypted folder in the Documents Library.
